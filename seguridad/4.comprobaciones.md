## Ver los loggins creados (Estar con el usuario windows o admin que tenga todos los permisos)

```sql
SELECT 
    name AS [LoginDelServidor], 
    type_desc AS [TipoDeLogin],
    create_date AS [FechaCreacion]
FROM 
    sys.server_principals
WHERE 
    type IN ('S', 'U', 'G') -- S=Login SQL, U=Usuario Windows, G=Grupo Windows
    AND name NOT LIKE '##%'; -- Filtra logins del sistema
```

## Ver al usuario dentro de la base de datos y el login a nivel servidor

```sql
SELECT 
    SUSER_SNAME() AS [LoginDelServidor (SS)],
    USER_NAME()   AS [UsuarioEnLaBD],
    DB_NAME()     AS [BaseDeDatosActual]
```
