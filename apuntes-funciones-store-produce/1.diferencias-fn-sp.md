## Función (User-Defined Function - UDF)

Es bloque de código que **obligatoriamente debe devolver un valor**, piensa en ella como una **calculadora** o una fórmula de Excel personalizada.

Funciones Escar (Scalar function)

-  Devuelve un *único valor* (un número, una cadena de texto, una fecha).

 **Características:**
   * No puede modificar datos.
   * Se puede usar directamente dentro de un `SELECT` o `WHERE`, como si fuera una columna más.
 
 **Ejemplo Sencillo** Una función que toma el nombre y apellido, puede devolver el nombre completo.


## Procedimiento Almacenado (Stored Procedure)

Es un conjunto de una o más instrucciones SQL (T-SQL).
Piensa en él como un **pequeño programa o una receta**

 **Características:**
  - **Modifica Datos:** Puede (y suele) hacer `INSERT`, `UPDATE`, `DELETE`.
  - **Transacciones:** Puede manejar transacciones explícitas (`BEGIN TRAN`, `COMMIT`, `ROLLBACK`).
  - **Cómo se usa:** Se llama con `EXEC` o `EXECUTE`.
